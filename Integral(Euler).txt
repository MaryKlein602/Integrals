Solution for differential equation with Euler method

The equation is: y + 3y / x = 4x
The steps are: h = 0,1, h = 0,2
The points are: y(1) = 3, so x = 1

So the x belongs to [1, 1,4]
The solution is: y(0) = ( x**5 + 2) / x**3

So the values are: x = [1, 1.1, 1.2, 1.3, 1.4]
                   y = [3.0, 2.712629601803155, 2.597407407407408, 2.600332271279017, 2.688862973760933]
